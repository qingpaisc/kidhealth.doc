## 列表

```
GET /wechat/v1/kid
```

## 查看

```
GET /wechat/v1/kid/:id
```


## 更新资料

```
Patch /wechat/v1/kid/:id
```

## 删除

```
Delete /wechat/v1/kid/:id
```

## 绑定儿童
```
GET /wechat/v1/kid/binding
```
> 参数 姓名:name; 身份证号:idcard;

## 获取儿童眼底报告列表
```
Get /admin/kid/newborn-report/{kidId}
```
## 获取儿童眼底报告详情
```
Get /admin/kid/newborn-report/detail/{reportId}
```
