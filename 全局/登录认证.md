# 登录认证

## 筛查客户端登录
`
POST /auth/v1/station
`

> 请求参数: 

<table>
    <tr>
        <th style="width:150px;">请求参数</th>
        <th>必须</th>
        <th>说明</th>
    </tr>
    <tr>
        <td>username</td>
        <td>是</td>
        <td>帐号</td>
    </tr>
    <tr>
        <td>password</td>
        <td>是</td>
        <td>密码</td>
    </tr>
</table>

> 返回值

<table>
    <tr>
        <th style="width:150px;">返回参数</th>
        <th>类型</th>
        <th>说明</th>
        <th>备注</th>
    </tr>
    <tr>
        <td>user</td>
        <td>object</td>
        <td>用户</td>
        <td>-</td>
    </tr>
    <tr>
        <td>token</td>
        <td>string</td>
        <td>TOKEN,后续请求需要使用</td>
        <td>-</td>
    </tr>
</table>
